# 全球休闲游戏个性化策略与调研方案

> **来源**：由 Canvas `global-casual-game-research.canvas.tsx` 转换。  
> **定位**：面向全球市场发行休闲游戏时，用于制定跨地区、跨文化、跨年龄和跨性别的用户研究与个性化策略。  
> **核心原则**：不要按国家、种族或性别硬编码玩家体验。全球化个性化应采用 **文化/地区假设 + 行为画像 + 明示偏好 + 公平合规约束** 的组合。

---

## 1. 研究支撑

| 研究方向 | 代表来源 | 关键结论 | 策略启发 |
|----------|----------|----------|----------|
| SDT 自我决定理论 | Ryan, Rigby & Przybylski；HCI Games SDT review | 自主感、胜任感、关系感驱动持续参与 | 用可选择难度、清晰反馈、可解释进步和轻社交满足不同人群 |
| 玩家动机模型 | Yee；Trojan Player Typology；Digital Games Motivation Scale | 成就、完成、社交、沉浸、竞争等动机跨文化存在差异 | 把玩家分为行为画像，而不是只用国家或性别做静态分群 |
| 文化维度与玩法 | Hofstede 相关跨文化游戏行为研究 | 国家可作为弱文化代理；个体主义、合作、竞争会影响玩法偏好 | 先形成地区假设，再用遥测和问卷验证，避免文化刻板化 |
| 本地化 / Culturalization | Game localization and culturalization research | 本地化不只是翻译，也包括符号、禁忌、节奏、支付与社交语境 | UI 文案、图标、节日、广告/IAP 提示要区域化审核 |
| 年龄与可访问性 | Older adult mobile game UX；children/older adult co-design | 年龄影响视觉、操作负担、奖励理解和社交需求 | 提供大字号、高对比、低操作负担、短会话与清楚反馈 |
| 游戏自博弈与评估 | TD-Gammon、AlphaGo/Zero、AlphaZero、MuZero、OpenSpiel | 自博弈和固定评估协议可验证规则是否可学、可玩、可复现 | Bot/RL 用于规则评估，不直接替代真人体验判断 |

## 2. 地区与文化策略假设

| 地区 | 文化/市场假设 | 体验策略 | 需要验证 |
|------|----------------|----------|----------|
| 北美 | 多元、个人选择强、隐私敏感、订阅/IAP 成熟 | 强调自主选择、成就路径、隐私控制、可跳过广告 | 短会话留存、挑战任务、可解释付费价值、GDPR/CCPA 风格合规 |
| 西欧 / 北欧 | 合规与公平敏感，低容忍误导式变现 | 降低诱导付费感，突出公平、可持续、无压力玩法 | 默认弱商业化、清晰同意、可关闭个性化、可访问性优先 |
| 东亚 | 完成、收集、排行、节奏优化常见，但国家差异很大 | 强化收集图鉴、连击、多消、排行榜和精细难度曲线 | 中国/日本/韩国分别做本地文化、支付、监管和活动验证 |
| 东南亚 | 移动优先、社交传播强、设备性能差异大 | 轻包体、弱网可玩、分享挑战、低端机画质档位 | 离线 MVP、社交邀请、广告频控、本地语言优先级测试 |
| 拉美 | 移动娱乐和社交传播强，价格敏感度较高 | 强化免费进度、广告奖励、社区挑战、节日活动 | 低价 IAP、奖励广告价值验证、WhatsApp/社媒分享素材测试 |
| 中东 / 北非 | 文化与宗教符号敏感，移动增长快 | 保守符号审核、家庭友好表达、RTL/阿语质量 | 内容 culturalization、斋月/节日节奏、支付可用性研究 |
| 印度 / 南亚 | 语言多样、低端设备和价格敏感并存，竞技/社交传播强 | 轻量安装包、本地语言分层、免费奖励、低网延迟 | 地区语言优先级、广告频控、低价包、设备性能队列测试 |
| 非洲重点市场 | 设备/网络/支付差异大，移动优先 | 极低资源模式、离线可玩、短会话、广告谨慎 | 包体、耗电、弱网、支付可达性与本地社区调研 |

## 3. 社会人口学特征与设计含义

| 分层 | 常见研究发现 | 可采用策略 | 注意事项 |
|------|--------------|------------|----------|
| 儿童 / 青少年 | 学习快、反馈敏感、家长/监管约束强 | 更清晰的正反馈、低挫败、无诱导付费、家长保护 | 年龄门槛、隐私保护、广告限制、未成年人合规 |
| 18-24 | 移动高频、跨平台、挑战和社交传播较强 | 快节奏挑战、排行榜、连击、多消、短视频分享素材 | 按行为而非年龄推难度，避免过度刺激 |
| 25-44 | 碎片时间、减压和进度感并重 | 短局、每日任务、稳定成长、可中断恢复 | 会话长度、通勤/睡前时段、奖励广告接受度研究 |
| 45+ | 可读性、低操作负担、熟悉规则和成就感更重要 | 大字号、高对比、慢节奏、明确提示、低惩罚 | 触控误差、视力、听力、认知负荷和教程理解测试 |
| 女性玩家 | 移动休闲和社交/进度动机常见，但个体差异大 | 温和竞争、装饰/收集、轻社交、无羞辱失败反馈 | 不要用性别硬编码；用实际行为画像调整体验 |
| 男性玩家 | 挑战、竞争、成就动机在部分研究中更强 | 挑战任务、排行榜、效率优化、多消/清屏目标 | 避免默认更高难；以表现和偏好确认 |
| 少数族裔 / 移民社群 | 语言、文化符号、代表性与社区语境重要 | 本地化文本、包容性视觉、文化禁忌审核 | 敏感属性不用于个人定向；只做聚合研究和包容性审查 |

## 4. 个性化数据边界

| 等级 | 变量 | 允许用途 |
|------|------|----------|
| 允许用于个性化 | 实时行为：失败率、思考时长、清行率、会话长度、主动选择、设备性能、语言/地区设置 | 动态难度、提示强度、画质、教程、奖励节奏、候选块概率 |
| 仅用于研究分层 | 年龄段、性别、国家/地区、文化背景、支付区间、设备档位 | 分析差异和设计假设，不直接决定个体待遇 |
| 敏感或高风险 | 种族、民族、宗教、健康、未成年人状态、精确位置、收入推断 | 默认不做个体级策略；仅在合法、必要、聚合、匿名时做包容性评估 |
| 必须透明 | 个性化难度、广告频控、IAP 提示、推送召回 | 提供说明、关闭入口、数据导出/删除和未成年人保护 |

## 5. OpenBlock 风格策略映射

| 玩家画像 | 识别信号 | 策略动作 | 心理目标 |
|----------|----------|----------|----------|
| 新手保护 | 低失败率、长思考、低清行 | 小块、保消、强提示、低压力、明确教程 | 胜任感，不让首局被随机死局劝退 |
| 高手挑战 | 高连击、高规划、低失误 | 顺序刚性、多消窗口、排行榜、限步挑战 | 能力匹配，避免无聊 |
| 减压玩家 | 短会话、低竞争、睡前/通勤场景 | 温和节奏、少弹窗、稳定奖励、柔和视觉 | 满足放松，不制造焦虑 |
| 收集/完成玩家 | 高任务完成率、回访稳定 | 图鉴、每日轻目标、可预期奖励、里程碑 | 完成感和长期目标 |
| 社交/比较玩家 | 分享、排行、邀请、挑战参与 | 异步挑战、好友榜、可分享高光，不强制社交 | 关系感和自我表达 |

## 6. 调研执行方案

| 阶段 | 方法 | 产出 |
|------|------|------|
| 1. 桌面研究 | 论文、市场报告、商店评论、竞品活动、地区监管 | 国家/地区假设库、风险清单、文化审核表 |
| 2. 定性研究 | 6-8 个重点市场，每市场 8-12 人访谈 + 可用性测试 | 教程理解、压力点、图标语义、奖励接受度、文化禁忌 |
| 3. 定量问卷 | 每市场 300-500 样本，覆盖年龄/性别/设备/语言 | 动机量表、付费/广告态度、会话场景、难度偏好 |
| 4. 遥测实验 | 软启动 A/B：难度、提示、广告频控、奖励节奏、画质 | D1/D7、失败后留存、会话长度、死局率、清行率、投诉率 |
| 5. 模型校准 | 用行为数据训练玩家画像，不把敏感属性作为个体决策输入 | 按行为聚类的个性化策略和公平性审计报告 |
| 6. 上线门禁 | 地区 checklist、合规审核、文化审核、可访问性、低端机测试 | 发布 Go/No-Go 和 post-launch 监控面板 |

## 7. 落地门禁

每个国家/地区上线前至少完成：

- 语言与文化审核。
- 低端机性能测试。
- 广告/IAP 合规检查。
- 未成年人保护。
- 可访问性检查。
- D1/D7 留存与死局率监控。
- 个性化关闭入口。

研究结论用于形成可验证假设；最终策略以当地数据、用户同意和公平性审计为准。

## 8. 当前游戏改进项与算法落地矩阵

| 方向 | 已落地字段/模块 | 策略含义 | 合规边界 |
|------|-----------------|----------|----------|
| 行为画像 | `PlayerProfile.behaviorSegment`：新手保护、高手挑战、减压、收集完成、社交比较、均衡 | 用真实玩法行为决定长期策略，不按国家/性别/种族硬编码 | 只用实时行为和明示偏好 |
| 动机意图 | `motivationIntent`：胜任、挑战、减压、收集、社交、均衡 | 与 `spawnIntent` 分工：前者描述中长期玩家动机，后者描述本轮出块意图 | 可关闭个性化难度 |
| 回归暖启动 | `returningWarmupStrength` | 沉默 1/3/7 天后短期减压、偏小块、提高保消，避免首局挫败 | 基于本地会话时间，不做推送定向 |
| 可访问性救济 | `accessibilityLoad` | 低画质/低动态/误触/长思考触发低压和操作负担降低 | 不推断年龄或健康状况 |
| 社交公平挑战 | `socialFairChallenge` | 异步挑战、固定 seed 模式关闭个体化难度，保证规则公平 | 个体画像不影响比赛结果 |
| RL 训练 | replay outcome-only、teacher metrics、beam risk adaptive、bonus auxiliary、dual eval | 提升算法可学性和评估可解释性 | 用于规则和 bot 评估，不替代真人体验判断 |
