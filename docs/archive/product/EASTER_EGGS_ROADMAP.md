# 彩蛋 / 惊喜路线图

> **来源**：由 Canvas `easter-eggs-roadmap.canvas.tsx` 转换。  
> **定位**：基于 OpenBlock v10.14 现状盘点和休闲游戏类比，给出彩蛋、惊喜反馈、皮肤资产放大和社交传播的候选方案。

---

## 1. 总览

核心玩法中的 bonus、combo、perfect、streak、关卡、复活已扎实。36 款皮肤是巨大但尚未充分利用的资产。当前空白主要集中在 7 个象限：

- 感官层。
- 皮肤微动效。
- 节日 / 时段换皮。
- 主动道具。
- 隐藏彩蛋。
- 签到经济。
- 社交分享。

| 指标 | 数值 |
|------|------|
| 候选项目 | 35 |
| P0 高 ROI | 11 |
| P1 中等 | 14 |
| P2 长期 / 大工程 | 10 |

## 5. 优先级执行建议

1. 先做 P0 高感知低成本项：程序化音效、设备震动、皮肤切换转场、节日推荐、签到面板、隐藏皮肤。
2. 再做资产放大项：皮肤环境粒子、皮肤专属音色、艺术化分享海报。
3. 最后做高工程量项：异步 PK、BGM、天气感知、长期社交系统。

---

## 6. 风险与护栏

- 所有动效必须接入 `prefers-reduced-motion`。
- 音效、震动、粒子需要提供强弱或关闭选项。
- 节日和文化元素必须经过本地化审核，避免误用宗教或地域符号。
- 道具经济不要破坏公平感，尤其是排行榜和挑战模式。
- 彩蛋应增强惊喜，不应阻断核心玩法。
